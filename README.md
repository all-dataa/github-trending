# project
全自动的 github trends 爬虫邮件发送

# how to use
## linux deploy
进入虚拟环境 ```source [yourenv]/bin/activate```

将程序挂在后台运行  ```nohup python app.py > output.log 2>&1 & ```

查看进程运行成功  ```ps aux | grep python```

关闭进程  ```kill [进程id]```
